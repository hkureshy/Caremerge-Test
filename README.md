# Caremerge Coding Challenge  

## Steps to test  

```js
$ git clone https://github.com/hkureshy/caremerge-test.git
$ cd caremerge-test
$ npm install
```  

### To test the execution with plain node.js callbacks
`$ node index.js`  

### To test the execution with async.js
`$ node async.js`  

### To test the execution with RSVP
`$ node rsvp.js`
